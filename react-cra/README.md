## 🚀 Quick start

1.  **Yarn Install**

    ```shell
    yarn
    ```

2.  **Local test**

    ```shell
    yarn start
    ```

    브라우저에서 `http://localhost:3000`을 열어 결과를 확인하세요.
    app/page.tsx를 수정하여 페이지 편집을 시작할 수 있습니다. 파일을 편집하면 페이지가 자동으로 업데이트됩니다.

3.  **build**
    ```shell
    yarn build
    ```
